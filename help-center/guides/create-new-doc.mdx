## Add a Documentation Page

<Steps>
  <Step title="Open your project in xeyla">
    Navigate to your project from the dashboard.
  </Step>
  <Step title="Click 'Add Page'">
    Find the **Add Page** button in the sidebar.
  </Step>
  <Step title="Enter details">
    Add a title and optional description for the new doc.
  </Step>
  <Step title="Save and edit">
    Click **Save** to create the page. Begin editing with the built-in editor.
  </Step>
</Steps>
