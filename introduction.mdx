## Overview

Xeyla provides a centralized space to organize, manage, and publish all your project documentation. You create structured docs, collaborate with teams, and customize layouts using MDX-powered pages. Start by setting up your workspace and build pages with components like cards, steps, and tabs for engaging content.

<Callout kind="info">
Xeyla supports version control, search, and analytics to keep your docs up-to-date and accessible.
</Callout>

## Getting Started

Follow these steps to create your first documentation page.

<Steps>
  <Step title="Create Workspace" icon="home">
    Sign up at `https://xeyla.com` and create a new workspace for your project.
  </Step>
  <Step title="Add First Page" icon="plus">
    Click "New Page" and select the `introduction.mdx` template.
  </Step>
  <Step title="Edit with MDX" icon="edit-3">
    
````mdx
## Welcome

Start editing this page with your content.

<Callout kind="tip">
Add components for better engagement.
</Callout>
````

  </Step>
  <Step title="Publish" icon="globe">
    Preview changes, then click "Publish" to make it live.
  </Step>
</Steps>

## Example Integration

Embed Xeyla docs in your app using our API.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.xeyla.com/v1/docs/{workspaceId}', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`
    }
  });
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  import requests

  headers = {'Authorization': f'Bearer {YOUR_API_KEY}'}
  response = requests.get('https://api.xeyla.com/v1/docs/{workspaceId}', headers=headers)
  docs = response.json()
  print(docs)
  ```
</CodeGroup>
